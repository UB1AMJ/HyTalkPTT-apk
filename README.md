# HyTalkPTT-apk
Для желающих я положу собранное приложение, с репозитория: @chepil/HyTalkPTT

### Подробнее:
Приложение для активации кнопки РТТ на телефонах:
- Device: Motorola LEX F10
- Android Version: Android 5.1.1 (API Level 22)

Чтобы можно было на этих телефонах использовать следующее програмое обеспечение:
- Target Application: HyTalk (com.hytera.ocean)

### Инструкция:
Для использование установите себе на телефон одно из приложений eng/rus версии и настройте сам телефон.
#### Инструкция со страницы проэкта HyTalkPTT: 
> Перед использованием этого приложения необходимо настроить две системные параметры:
> 
> 1. Программируемые клавиши
> Перейдите в Настройки → Программируемые клавиши
> Выберите Приложение PTT Key
> Выберите HyTalkPTT
> Это позволяет приложению получать сигналы от кнопки PTT, когда устройство заблокировано или приложение работает в фоновом режиме.
> 
> 2. Служба специальных возможностей
> Перейдите в Настройки → Специальные возможности
> Найдите HyTalkPTT в списке
> Включите переключатель
> Это даёт приложению разрешение на глобальный перехват ключевых событий.
